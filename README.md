1- What is the difference between var, let, and const?
Ans: 
var: A variable declared with var can be updated and redeclared.
let: A variable declared with let can be updated but cannot be redeclared in the same scope.
const: A variable declared with const cannot be updated or redeclared.

2- What is the spread operator (...)?
Ans: The spread operator (`...`) in JavaScript is used to expand or copy elements of an array or object into another array or object.

3- What is the difference between map(), filter(), and forEach()?
Ans:
map(): Creates a new array by applying a function to every element of an array.
filter(): Creates a new array with elements that pass a condition.
forEach(): Loops through each element of an array but does not return a new array.

4- What is an arrow function?
Ans: An arrow function is a short and modern way to write a function in JavaScript using `=>` syntax.

5-  What are template literals?
Ans: Template literals are a way to create strings in JavaScript using backticks (`) instead of quotes, allowing interpolation and multiline strings
